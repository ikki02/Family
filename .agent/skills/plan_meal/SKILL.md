# 献立を考える
Notion MCP経由で直近の「献立」を読み取って、おすすめの献立を提案してください。

## 手順
- Notion MCP経由で直近の「献立」を読み取る
  - 指定がなければ、直近2週間分を読み取る
- 直近の献立の主菜について、肉類、魚類、野菜、の傾向を振り返る
- もし使いたい食材の指定があれば、それを優先する。
  - 例: キャベツ、大根を使いたい、など。
- なるべく同じ傾向が続かないように、提案する献立のバランスを考える
- 1ヶ月以上前の献立は参照して返答に含めても良い
- 3週間以上前の⭐️がついたレシピは1つ含めても良い

## 出力の例
2/9昼: 豚肉の生姜焼き  
2/9夜: 鮭の塩焼き  
2/10昼: 鶏肉の唐揚げ  
2/10夜: 牛丼
